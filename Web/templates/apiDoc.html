API - Documentación

Volver

classesNumber

Cuenta el numero de clases resultantes de una serie de datos.

Endpoint

http://localhost:5000/api/classesNumber/{typo}

Typo

  • 0: Calculado desde un conjunto de datos
  • 1: Calculado desde la longitud de los datos

Payload - typo 0

Datos enviados por Body en JSON.

[1, 2, 3, 4, 5, 6]

Response - typo 0

4

Payload - typo 1

Datos enviados por Body en JSON.

6

Response - typo 1

4

dataRange

Calcula el rango de los datos de una serie de datos.

Endpoint

http://localhost:5000/api/dataRange/{typo}

Typo

  • 0: Calcula a través de una serie de datos.
  • 1: Calcula a través del dato menor y el dato mayor de una serie de datos.

Payload - typo 0

Datos enviados por Body en JSON.

[1, 2, 3, 4, 5, 6]

Response - typo 0

5

Payload - typo 1

Datos enviados por Body en JSON.

[1,6]

Response - typo 1

5

dataAmplitude

Calcula la amplitud de los datos por cada clase de un conjunto de datos.

Endpoint

http://localhost:5000/api/dataAmplitude/{typo}

Typo

  • 0: Calcula a través de una serie de datos.
  • 1: Calcula a través de un rango, numero de clases y unidad de variación.

Payload - typo 0

Datos enviados por Body en JSON.

[1, 2, 3, 4, 5, 6]

Response - typo 0

2

Payload - typo 1

Datos enviados por Body en JSON.

{
    "range": 5,
    "classes": 4,
    "variation": 1
}

Response - typo 1

2

variationUnit

Endpoint

http://localhost:5000/api/variationUnit

Payload

3

Frecuency

Calcula la frecuencia de una clase.

Endpoint

http://localhost:5000/api/frecuency

Payload

Datos enviados por Body en JSON.

{
    "data": [1, 2, 3, 4, 5, 6],
    "limInf": 0.5 ,
